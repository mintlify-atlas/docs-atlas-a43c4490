## Welcome to OI Resume App

OI Resume App is a Swift-based iOS application that elegantly showcases your professional skills and experiences. Built entirely with SwiftUI, it provides an interactive and modern way to present your portfolio with smooth animations and collapsible sections.

The app allows users to expand or collapse different sections—such as skills and experiences—through interactive buttons with animated chevrons, creating a clean and engaging user experience.

## Key features

<AccordionGroup>
  <Accordion title="SwiftUI-based architecture">
    Built entirely with SwiftUI, leveraging the latest iOS development framework for declarative UI design. The app uses modern SwiftUI patterns including `@State`, `@ObservableObject`, and combines views efficiently for optimal performance.
  </Accordion>

  <Accordion title="Interactive collapsible sections">
    Skills and experience sections can be expanded or collapsed with smooth animations. The chevron icons rotate 180 degrees with `.easeInOut` animations, providing delightful user feedback.
  </Accordion>

  <Accordion title="Dark and light mode support">
    Full support for iOS system appearance modes. The app automatically adapts to the user's preferred color scheme using `Color(UIColor.systemBackground)` and dynamic colors throughout.
  </Accordion>

  <Accordion title="XCUITest testing framework">
    Comprehensive UI testing using XCUITest with Page Object Model (POM) pattern for better maintainability. Tests verify section visibility, toggle functionality, and user interactions.
  </Accordion>

  <Accordion title="Custom typography">
    Beautiful Montserrat font family integration with multiple weights (thin, regular, medium, semibold, bold) and italic variants for a polished, professional appearance.
  </Accordion>

  <Accordion title="Modular component architecture">
    Clean separation of concerns with dedicated views: `HeaderView`, `SkillsView`, `ExperiencesView`, and their supporting components, making the codebase easy to maintain and extend.
  </Accordion>
</AccordionGroup>

## App structure

The OI Resume App follows a clean, organized structure:

```
OI_Portfolio/
├── Code/
│   ├── Controllers/
│   │   └── OI_PortfolioApp.swift      # Main app entry point
│   ├── Models/
│   │   └── Portfolio.swift            # Data models (Skill, Experience, Portfolio)
│   ├── Services/
│   │   └── AppModel.swift             # Portfolio data and business logic
│   └── Views/
│       ├── PortfolioView.swift        # Main view
│       └── SupportViews/
│           ├── HeaderView.swift       # Profile header
│           ├── SkillsView.swift       # Skills section with grid
│           ├── ExperiencesView.swift  # Experience section
│           ├── SkillView.swift        # Individual skill card
│           └── ExperienceView.swift   # Individual experience item
├── Extensions/
│   └── ColorExtensions.swift          # Hex color support
└── Utilities/
    └── FontManager.swift              # Montserrat font management
```

<Note>
  The app uses the Page Object Model pattern for UI testing, separating test logic from page interactions for better maintainability and reusability.
</Note>
